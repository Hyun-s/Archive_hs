# Personal Portfolio Website

This is my personal portfolio website featuring CV and calendar functionality.

## Features
- Responsive design that works on all devices
- Interactive CV timeline with professional experience
- Functional calendar with event indicators
- Modern UI with smooth animations and transitions
- Clean, professional aesthetics

## Deployment
This site is automatically deployed via GitHub Pages from the `main` branch.

## Technologies Used
- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript ES6+
- Font Awesome icons
- Google Fonts (Inter)

## Access
The portfolio is live at: https://Hyun-s.github.io/Archive_hs/

## Development
To develop locally:
1. Clone this repository
2. Serve the files using any local HTTP server (e.g., `python3 -m http.server 8000`)
3. Make changes to the HTML/CSS/JS files
4. Push changes to GitHub to update the live site
